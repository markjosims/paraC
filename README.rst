Morphological Analysis with Paradigms and Rules
===============================================

Other name ideas:
- MOrphological DOcumentation with Paradigms And Rules (MODOPAR)

Morphological Analysis with Paradigms and Rules (MAPR, pronounced "mapper") is a tool for creating automatic morphological parsers.
Following the name, MAPR is designed on two principles: words are organized into inflectional paradigms, and inflectional processes are represented with context-sensitive rules (in addition to affixation).
MAPR aims to allow the creation of morphological parsers with a minimal learning curve.
Where previous tools (xfst, hfst, Foma) require learning a domain-specific language and the underlying math of Finite State Transducers, MAPR provides an intuitive user interface built around concepts familiar to linguists (e.g. morphological features, lexical diacritics, principal parts).

MAPR is built on the ``pynini`` Python package (Gorman 2016).
