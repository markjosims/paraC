# Constants
This directory includes several submodules defining constants used elsewhere in the parser.
- `features.py`: Defines morphological features for various parts of speech and flags indicating lexical information used to aid in parsing and glossing.
- `paths.py`: Defines filepaths for various program data.
- `search_constants.py`: Defines parameters related to fuzzy search.
- `symbol_table.py`: Defines symbol tables used by FST parsers, including Tira phone inventory and various special symbols.
